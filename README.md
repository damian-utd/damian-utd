## Featured project

### Dietitian Support System

A fullstack web application for dietitians, created as my engineering thesis project. The system helps manage patients and create personalized nutrition plans based on patient data.

Main features:

* user registration and authentication,
* JWT-based authorization stored in HttpOnly cookies,
* patient management,
* nutrition plan creator,
* food product search based on a nutritional database,
* BMI, BMR and TDEE calculations,
* PostgreSQL database integration,
* transactional saving of diet plans,
* multi-container Docker environment.
